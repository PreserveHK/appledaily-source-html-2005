乜都免費！商場大使激變制服幫

「今時今日，咁嘅服務態度係唔夠嘅！」所以好多商場都盡量做到最好。除咗顧客服務內容列明的免費着數服務，更提供青春無敵、身懷絕技的「流動服務大使」，令你賓至如歸，唔怕你唔食回頭草返來幫襯。今期佢哋重潮到扮埋Lolita，問你死未？

記者：邵超
攝影：黃子偉、陳盛臣
模特兒：Hoston、Vanessa（Salvador）

我哋係流動服務大使！

着數1.流動大使助人團

流動服務大使是近幾年在商場崛起的一幫人物，四圍巡場主動去幫人。幫人係一個非常抽象的概念，點樣幫？幫到咩程度？借錢當然唔得！只要係可以用手用腳用到把口嘅，只要你開句聲，他們點都幫你幫到底。當你一臉茫然，他們會失驚無神出現幫你解決問題，快過「及時雨」宋江，擔得抬得勁過阿四。

幫人個案一：落場也要幫
Cathy＠朗豪坊：「一位英文唔流利的日本太太問路，遇着我放食晏，於是親自帶她去女人街，她向我鞠躬多謝，當時好開心。」

幫人個案二：外國人都幫
Benny@朗豪坊出手幫忙後，受惠人BronteDavies說：「我喺澳洲嚟香港旅行，唔識去銅鑼灣，他的英文好好，解釋得好清楚，好滿意他的服務。」結果Benny更送埋贈品——旅遊發展局印製的旅遊地圖一張。

幫人個案三：學手語去幫
Creamy@德福廣場：「同事之間成日報班鬥攞得多Certificate，現在我上緊手語中班，有次用手語溝通，幫咗一個腳抽筋的聾啞人士，原來佢只係需要一杯熱水和一張櫈。」

着數2.免費影印＋Fax

喺港九新界都擁有一個流動辦公室，隨時上網處理公務、私務，附送私人助理，幫你手影印、傳真、寄信、餐廳訂位*、訂花、花籃*，費用全免！邊度有咁大隻蛤乸隨街跳？都係以下呢啲商場幫到你。
Checkpoint
地鐵商場、國金、海港城、時代廣場、旺角中心等
（*兩項服務只限信和集團旗下商場）

着數3.免費寄存戰利品

瘋狂購物最怕提購物袋，原來唔少商場都提供免費物品寄存服務，只要在商場關門前取回便可。最特別是國金，顧客購物後，商舖會致電商場的禮賓部，「代客提送服務大使」便會於10分鐘內到達該店舖，代客提送禮品到商場的指定地點或暫存「戰利品」至購物完畢。
Checkpoint
國金、apm、朗豪坊、新世紀廣場、海港城、奧海城等

着數4.免費手機相機充電

早前我哋介紹過家陣7-Eleven便利店，只須付$8即享快速充電服務。但原來喺商場，呢啲手機相機叉電服務竟然費用全免，叉電叉到飽為止。
Checkpoint
奧海城、旺角中心、海港城、時代廣場、地鐵商場、新世紀廣場等

着數5.免費聽歌睇雜誌

許多人都留意到商場有免費音樂試聽，但就忽略咗鄰台的免費雜誌。原來喺apm提供的係最Update的日本潮流雜誌嚟㗎！
Checkpoint
apm

着數6.免費無線上網

同部份地鐵站一樣，喺商場一樣要企定定無櫈坐使用無線上網服務。
Checkpoint
海港城、時代廣場、地鐵商場、新世紀廣場、apm等

着數7.借遮唔使錢

近排失手唔記得帶傘？唔緊要！好多商場都有提供免費雨衣及雨傘借用服務（需付按金，七天內歸還）。
Checkpoint
海港城、地鐵商場、新世紀廣場等

着數8.免費擦鞋機

擦鞋機價值三萬元，幫男士自動擦靚黑色及咖啡色皮鞋。唔識用？搵服務大使幫你啦！
Checkpoint
德福廣場、國金等

着數9.防偽點鈔機

雖然一般人好難用到，不過商場內的商戶職員反而常用，始終係一個免費幫到人的服務。
Checkpoint
海港城、時代廣場等

着數10.保母出動

合家歡商場通常有提供免費紙尿片、嬰兒車借用及提供冷熱水及嬰兒食品翻熱。其中海港城更設兒童遊樂區，作為購物小休地點。
Checkpoint
地鐵商場、海港城、時代廣場、國金等
