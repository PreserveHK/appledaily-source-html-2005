數碼影音：
新春旅行平掃DC

AV場Sales報料，呢排啲數碼相機去貨極度凌厲，無他嘅！農曆新年就快到，好多人都諗住去拜年時，或趁呢個春節出外旅遊時，可以拍番啲靚相留念也！一般人買機要求，主要可分為拍Movie、輕巧易用或Prosumer三大要求，今日就教大家呢期減價DC全攻略，加埋近期記憶卡同數碼相簿價格回落，包你可以放開胸懷盡情影！

記者：朱崇德　攝影：陳盛臣　ImageGirl：Emily（D8）

提提大家，如果短期內想出機真係要手快！事關春節臨近，AV場一般都會做多啲「存倉貨」，而影響所及當然係落單較緊張兼補貨較難！所以下面所介紹嘅減價DC，貨量相信會較平時緊張，唔想等買「補貨」，切記一定要快手！

輕巧好拍

RICOHR1v28mm廣角鏡

R1v是剛落場人氣之機，僅厚25mm，纖巧型DC中，佢擁有罕見28mm廣角鏡，加上500萬像素CCD成像極滑淨，所以成為了好多專業攝影師的「口袋機」！

R1v反應迅速；開機時間為0.8秒，快門反應為0.003秒！由開機至影完相都唔使1秒，絕對是最強SnapShot機王！電源使用較容易補充的AA電池，但R1v在開芒、用閃燈又睇相的情況下，竟可拍下近500張相片！慳電能力可用恐怖形容。

Profile
CCD：500萬像素（1/2.5")
鏡頭：4.8×光學/3.4×數碼
記憶卡：SD、MMC

CASIOEX-S100纖巧機王

EX-S100是現時最炙手可熱Slim機王，使用最新透明陶瓷鏡頭；拍攝反應亦快，快門遲緩時間僅1秒，亦可拍320×240/15fps無時限MotionJPEG。

Profile
CCD：300萬像素（1/3.2")
鏡頭：2.8×光學/4×數碼
記憶卡：SD（內建9.8MB）

SONYT33人氣新機

T33全金屬機身，一樣Sell薄身加2.5茪j芒；不過擁有510萬像素影相能力，而Movie模式最高有640×480/30fps，要慳記憶體亦有160×112/8.3fps。

Profile
CCD：510萬像素（1/2.4")
鏡頭：3×光學/6×數碼
記憶卡：MSDuo、MSProDuo

Movie狂熱分子

SANYOC4Movie最強！

C4採用400萬像素CCD，經插值，最高可輸出800萬像素相片；最強是它的MPEG4攝錄功能，影像質素共分三種，分別為TV-SHQ（640×480/30fps/3Mbps）、TV-HQ（640×480/30fps/2Mbps）及TV-S（320×240/30fps），頭兩種都達DVD質素，最低影像格式亦達VCD水平；使用1GBSD卡情況下，可分別錄下41分鐘、1小時及2.5小時的MPEG4影像片段；此外C4更擁有DC界首部「二聲道」收音功能，而且又可以拍住Moive、邊影相，再加上光學防手震功能！簡直同部DV冇分別！

Profile
CCD：400萬像素（1/2.7")
鏡頭：5.8×光學/10×數碼
記憶卡：SD

PENTAXMX4370mm大炮王

雖是舊款機種，但一樣玩MPEG4錄像（640×480/30fps），最強係佢支大炮有10×光學變焦（37-370mm），暫時無人能及，影像特別清晰細緻！

Profile
CCD：400萬像素（1/2.7")
鏡頭：10×光學/10×數碼
記憶卡：SD

SANYOVPC-J4DCDV兩手準備

J4一樣使用400萬像素CCD，可插值上800萬像素，備有12種場景模式；拍片以MotionJPEG格式，解析度為640×480/30fps。

Profile
CCD：400萬像素（1/2.7")

Nikon8400對焦超準

COOLPIX8400是Prosumer機中少見擁有24mm超廣角鏡頭；CCD使用特大號2/3?800萬像素CCD，新加的存相模式有RAW及低壓縮比的JPEG（Extra），成像特別滑淨！機背保留多角度扭擰芒傳統！尺寸亦由前代的C5400的1.5蚍W大至1.8荂A睇相更正！整體操控方面，反應感覺迅速，而且對焦「中」嘅感覺，差不多拍得住D-SLR機，係呢期最強之Prosumer機。

外遊拍檔高容量電子相簿

ARCHOSAV480睇相兼娛樂

出外長途旅遊，多數都會買部數碼相簿作儲相；最近至HiT是ARCHOSAV480，硬碟容量達80GB，去一個月旅行都未必用得晒！每次叉電可存入2-4GB的相片，內置插卡位只對應CF，其他卡就需要使用CFAdapter；另外AV480重有睇相、播MPEG4電影及MP3功能。

MAGIC-PRODCPartner$1,100入門級

平平哋相玩電子相簿，可以考慮MAGIC-PRO嘅DCPartner，40GB容量，但可對應CF、MD、SD、MMC、MS及SM多款卡種，不過就減省了睇相及多媒體娛樂功能；內置電池每次叉電可維持1-2GB相片上傳，要勤啲叉電。
